Our Data

The data we selected cam from Kaggle.com. It was available in several different formats, namely CSV as well as JSON. The data was extensive, as it contained records from 1969 to present. As a result the files were large, so we opted to use the available API in order to streamline the retrieving and cleaning process. A sample of the raw data is shown below.

Data Cleaning Process

After looking at the data, there were several fields that were not needed or repetitive. Some of these superfluous fields included "report_link", "posted", and "text". Additionally, any records containing null values were then removed in order to filter out less thorough reports. A "year" column was added to the dataframe by using the "split" function to break down the provided datetime. This column was useful for sorting and other date-relevant controls on the display of data.
