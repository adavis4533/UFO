About this project

Data Information

The UFO dataset was found on Kaggle.com and had a lot of information that needed to be dropped. The dataset has information on most UFO sightings reported in the United States dating back to 1969.

Last updated 1 day ago

Cleaning the data

The data was cleaned using Python in Jupyter Notebook. Because the dataset was so large, we had to use the provided API rather than using a CSV or JSON file.

Last updated 1 day ago

Linking the Database

We used MongoDB to import the clean data and proceeded to link the database to the Jupyter Notebook.

Creating the Website
