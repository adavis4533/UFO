UFO Sightings Troubleshooting

As with most projects, there are a lot of unforseen issues that come along. Here is a brief glimpse into some of the issues and solutions utilized.

Data Extraction/Cleaning

  • Data provided was large csv or json file with over 100k records dating back to 1969
  • Direct import into mongo did not work properly due to long character strings in the "summary" section of data
  • CSV could not be shared on Git because it was too large, so more practical to use provided API in jupyter notebook
  • Initially connected jupyter to mongo to import clean data

Data Visualization/Mapping

  • Creating flask app connecting to mongo unsuccessful (we think due to the number of records)
  • Extracting from mongo did not provide a usable json in javascript
  • Ended up using "to_json" in jupyter and using that json for map
  • Additional difficulties getting d3 to do what we wanted it to do in the map

Rendering Our Project

  • When moving files from each computer into the repository, took time/effort to get arrangement/file paths correct
  • Initially tried to launch the homepage using Flask, but map would not render
  • Decided to stick to python server to ensure map functionality opposed to using flask
